## 1. เป้าหมายและกลุ่มผู้ใช้

**โมเดลธุรกิจของเว็บ:** เว็บไซต์ = แคตตาล็อก + สร้างความเชื่อมั่น → **ปิดการขายผ่านแชท** (LINE สำหรับคนไทย / WhatsApp–Facebook สำหรับต่างชาติ) ไม่มีตะกร้าสินค้า–ระบบจ่ายเงินในเฟสแรก

| กลุ่ม | พฤติกรรม | สิ่งที่เว็บต้องให้ |
|---|---|---|
| ลูกศิษย์/นักสะสมไทย | รู้จักอาจารย์อยู่แล้ว มาดูรุ่นใหม่ เช็คของหมด | อัพเดตเร็ว, ค้นตามอาจารย์, กดทัก LINE ได้ใน 1 คลิก |
| ผู้สนใจไทยหน้าใหม่ | เข้าจาก Google/YouTube สนใจกุมารทอง–เครื่องรางเสน่ห์ | บทความให้ความรู้ (วิธีบูชา คาถา), ความน่าเชื่อถือ |
| ต่างชาติ (SG/MY/ฯลฯ) | ไม่อ่านไทย ต้องการหลักฐานว่าของแท้–ร้านจริง | หน้า EN, ราคา, วิธีสั่ง–ส่งต่างประเทศ, ใบรับประกัน, รูปพิธี |

## 2. อารมณ์และทิศทางภาพ (Art Direction)

**คีย์เวิร์ด:** ขลัง · ศักดิ์สิทธิ์ · พรีเมียม · ทันสมัย — เหมือน "หอพระในโรงแรมหรู" ไม่ใช่บ้านผีสิง

- แรงบันดาลใจภาพ: แสงเทียน, ทองเปลวบนพื้นรักดำ, ลายยันต์เส้นทองบางๆ, ควันธูปจางๆ
- ใช้พื้นดำอมม่วง/น้ำตาลอุ่น (ไม่ใช่ดำสนิท #000) ให้ภาพสินค้าดูนุ่มและหรู
- ทองใช้ 2 น้ำหนัก: ทองหลัก (ปุ่ม/หัวข้อ) และทองอ่อน (เส้น/ลายประดับ)
- **หลีกเลี่ยง:** แดงเลือด, ฟอนต์สยองขวัญ, ภาพมืดทึบ, เอฟเฟกต์ผีๆ, ดาวเพนตาแกรม

## 3. โทนสี (Design Tokens)

| Token | Hex | ใช้กับ |
|---|---|---|
| `night` (พื้นหลัก) | `#100D14` | background หลัก — ดำอมม่วงอุ่น |
| `night-soft` (พื้นการ์ด) | `#1A1622` | การ์ดสินค้า, section สลับ |
| `gold` (ทองหลัก) | `#C9A227` | ปุ่มหลัก, หัวข้อ, ราคา |
| `gold-light` (ทองอ่อน) | `#E8C86A` | hover, เส้นขอบ, ลายยันต์ประดับ |
| `ivory` (งาช้าง) | `#F2ECDC` | ตัวหนังสือหลักบนพื้นดำ |
| `smoke` (ควัน) | `#9B93A6` | ตัวหนังสือรอง, caption |
| `ember` (เปลวเทียน — accent) | `#D96C3D` | badge "ใหม่", จุดเน้นเล็กๆ ใช้น้อยมาก |
| `line-green` | `#06C755` | ปุ่ม LINE เท่านั้น (สีแบรนด์ LINE ห้ามเพี้ยน) |

หลักการ: ทอง = ศักดิ์สิทธิ์/CTA หลัก, ember ใช้ไม่เกิน 5% ของหน้า, สถานะ Sold Out ใช้ smoke ไม่ใช้แดง

## 4. Typography

| บทบาท | ฟอนต์ | เหตุผล |
|---|---|---|
| หัวข้อไทย (Display) | **Prompt** (weight 500–600) | ทันสมัย เหลี่ยมสวย อ่านง่าย ไม่โบราณเกิน |
| เนื้อความไทย (Body) | **Anuphan** หรือ Noto Sans Thai (300–400) | อ่านสบาย รองรับ EN ในตัว |
| หัวข้อ EN (Display) | **Cormorant Garamond** (serif) | ให้ฟีล luxury/antique สำหรับตลาดต่างชาติ |
| ตัวเลข/SKU/ราคา | tabular-nums ของ body font | ตัวเลขตรงคอลัมน์ |

โหลดผ่าน `next/font/google` — ทุกตัวมีบน Google Fonts ฟรี

## 5. โครงสร้างเว็บ (Sitemap)

```
/                       หน้าแรก
/products               วัตถุมงคลทั้งหมด (filter: อาจารย์ / ประเภท / สถานะ)
/products/[id]          รายละเอียดสินค้า
/masters                รวมครูบาอาจารย์ (หน้าใหม่ — จุดขายหลักของร้าน)
/masters/[slug]         ประวัติอาจารย์ + วัตถุมงคลของท่าน
/articles               บทความ: คาถา วิธีบูชา พิธีกรรม
/articles/[id]          บทความเดี่ยว
/news                   ข่าวงานพิธี พุทธาภิเษก
/how-to-order           วิธีสั่งซื้อ (TH: LINE / EN: WhatsApp + ส่งต่างประเทศ)
/about                  เกี่ยวกับร้าน + ใบรับประกัน + รูปพิธี (trust page)
/en/*                   เวอร์ชันอังกฤษ (เฟส 2)
```

จัดหมวดสินค้าใหม่จาก 40+ หมวดของ igetweb เหลือ 2 แกน:
- **แกนอาจารย์** — อ.สุบิน นะหน้าทอง, หลวงปู่กาหลง, พระครูปืน ฯลฯ
- **แกนประเภท** — กุมารทอง (บูชา/พกพา), เครื่องรางมหาเสน่ห์, เสริมโชคลาภ, สะเดาะเคราะห์
- สถานะ (พร้อมเช่า / หมดแล้ว) เป็น filter ไม่ใช่หมวด

## 6. โครงหน้าหลัก ๆ

### หน้าแรก
1. **Hero** — ภาพวัตถุมงคลเด่นบนพื้นดำ + headline "วัตถุมงคลสายตรงจากครูบาอาจารย์" + CTA ทอง "ดูวัตถุมงคล" + ปุ่ม LINE
2. แถบความเชื่อมั่น — ✓ ของแท้จากพิธีจริง ✓ ใบรับประกัน ✓ ส่งทั่วโลก
3. รุ่นใหม่ล่าสุด (product grid 4–8 ชิ้น)
4. เลือกตามอาจารย์ (การ์ดรูปอาจารย์แนวนอน เลื่อนได้)
5. เลือกตามประเภท (กุมารทอง / เสน่ห์ / โชคลาภ)
6. บทความล่าสุด 3 ชิ้น
7. แถบวิธีสั่งซื้อ 3 ขั้น + LINE/WhatsApp

### หน้ารายการสินค้า
- Grid 2 คอลัมน์ (mobile) / 4 (desktop), filter ติดบน: อาจารย์, ประเภท, สถานะ
- ของหมดยังแสดง (สร้างความน่าเชื่อถือ+SEO) แต่จางลง + badge "หมดแล้ว" สี smoke อยู่ท้ายรายการ

### หน้าสินค้า
- ซ้าย: แกลเลอรีรูป (ซูมได้) / ขวา: ชื่อ, อาจารย์ (ลิงก์ไปหน้าอาจารย์), ราคา, SKU, ปุ่ม **"สอบถาม/สั่งซื้อทาง LINE"** (ส่งชื่อรุ่น+SKU เข้าแชทอัตโนมัติ)
- ใต้ลงมา: รายละเอียด, คาถาบูชา/วิธีบูชา, รุ่นอื่นของอาจารย์เดียวกัน

### หน้าอาจารย์ (ใหม่)
- ภาพ+ประวัติ+วิดีโอ YouTube พิธี → วัตถุมงคลของท่านทั้งหมด — หน้านี้คือความต่างจากร้านทั่วไป

## 8. สองภาษา (i18n)

- เฟส 1: ไทยล้วน โครงรองรับ i18n ไว้ (`next-intl`, route `/en`)
- เฟส 2: แปล UI ทั้งหมด + สินค้าเด่น ~30 รายการ + how-to-order ฉบับต่างชาติ (WhatsApp, ค่าส่ง EMS/DHL, ระยะเวลา)
- ราคาแสดง THB เป็นหลัก, EN แสดง USD โดยประมาณกำกับ

## 9. ความน่าเชื่อถือ (สำคัญมากสำหรับตลาดต่างชาติ)

- หน้า about: ประวัติร้าน, ใบรับประกันของแท้ (มีไฟล์ Adใบรับประกัน.jpg อยู่แล้ว), รูป/วิดีโองานพิธีจริง
- ทุกสินค้า: ระบุอาจารย์+พิธีที่ปลุกเสก
- ฝัง YouTube ช่องของร้าน
- รีวิว/ภาพจากลูกค้า (เฟสถัดไป)
